## 页面功能分析

### 实际功能

`providers.vue` 页面是一个**API提供商性能监控仪表盘**，提供以下功能：

1. **连接状态显示**
   - 显示是否连接到真实 Hermes Agent 数据库
   - Mock 模式标识（当数据库不可用时）

2. **性能统计卡片**
   - 平均响应时间
   - 成功率
   - 总调用次数
   - 活跃提供商数量

3. **提供商性能表格**
   - 提供商名称
   - 模型名称
   - 调用次数
   - 成功率
   - 平均响应时间
   - P95 响应时间
   - 状态徽章

4. **可视化图表**
   - 响应时间分布柱状图
   - 成功率趋势折线图（24小时）

5. **错误日志**
   - 最近失败的 API 调用
   - 错误详情和时间

6. **提供商详情模态框**
   - 点击表格行显示详细信息
   - 最近调用历史

### API 实现

**端点**: `GET /api/providers`

**数据来源**:
- 尝试连接 Hermes Agent 数据库（SQLite）
- 查询最近 7 天的 `sessions` 表数据
- 使用 `billing_provider` 字段统计提供商性能
- 如果数据库连接失败，返回 Mock 数据

**返回数据结构**:
```json
{
  "summary": { ... },
  "providers": [ ... ],
  "responseTimeDistribution": [ ... ],
  "successRateTrend": [ ... ],
  "errors": [ ... ],
  "isRealHermesConnected": true/false
}
```
