# Hermes Show 使用文档

Hermes Show 是 Hermes Agent 的 Web 可视化管理面板，提供直观的图形界面来管理和监控你的 AI 助手。

## 快速开始

\`\`\`bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:3000
\`\`\`

---

## 页面功能概览

| 页面 | 路由 | 功能 |
|------|------|------|
| 仪表盘 | \`/\` | 系统概览、Tokens消耗、活跃任务 |
| 配置中心 | \`/config\` | 模型、Agent、终端等配置管理 |
| 对话历史 | \`/history\` | 历史会话检索与查看 |
| Skills管理 | \`/skills\` | 技能库浏览与管理 |
| 定时任务 | \`/cron\` | 定时任务创建与管理 |
| 系统日志 | \`/logs\` | 实时日志查看与过滤 |
| 网关状态 | \`/gateway\` | 网关运行状态与平台连接 |

## 1. 仪表盘 (Dashboard)

**路由:** \`/\`

仪表盘是系统的主页面，提供 Hermes Agent 运行状态的全面概览。

### 功能模块

#### 连接状态
- 显示是否成功连接到本地 Hermes Agent
- 绿色状态：已连接 \`~/.hermes\` 目录
- 黄色状态：Mock 模式（未检测到配置目录）

#### 统计卡片
| 指标 | 说明 |
|------|------|
| 今日 Tokens | 当日 API 调用消耗的 Token 数量 |
| 总会话数 | 历史会话总数 |
| 今日会话 | 当日新增会话数 |
| CPU 负载 | 系统处理器使用率 |
| 活跃 Agent | 当前运行中的 Agent 数量 |
| API 延迟 | API 响应平均延迟 |

#### Tokens 消耗趋势图
- 展示近 7 天的 Token 消耗趋势
- 折线图可视化，支持鼠标悬停查看详情

#### 活跃任务
- 显示当前正在执行的任务列表
- 包含任务名称、Agent 类型、平台来源
- 进度条动画显示执行状态
- 点击可跳转到对应会话详情

#### 最近会话
- 表格形式展示最近的对话记录
- 显示会话标题、平台、时间、Tokens 消耗
- 点击行可跳转到会话详情

## 2. 配置中心 (Config)

**路由:** \`/config\`

配置中心用于管理 Hermes Agent 的各项设置，所有配置会保存到 \`~/.hermes/config.yaml\`。

### 功能模块

#### 模型配置
| 字段 | 说明 |
|------|------|
| 当前提供商 | 选择 API 提供商（OpenAI、Anthropic、自定义等） |
| 模型名称 | 指定使用的模型，如 \`gpt-4o\`、\`claude-sonnet-4\` |
| API 端点 | 提供商的 API 地址 |
| API 模式 | 调用模式（openai/anthropic） |
| 密钥环境变量 | API Key 所在的环境变量名 |

#### Agent 设置
| 字段 | 说明 | 默认值 |
|------|------|--------|
| Max Tokens | 单次回复最大输出 Token 数 | 8000 |
| Max Turns | 单次任务最大工具调用轮数 | 90 |
| Gateway Timeout | 网关超时时间（秒） | 1800 |
| Reasoning Effort | 推理强度 (low/medium/high) | medium |

**开关选项:**
- 流式输出 (Streaming): 启用实时流式响应
- 保存训练轨迹: 保存对话用于后续训练
- 详细日志 (Verbose): 输出更多调试信息

#### 终端配置
| 字段 | 说明 | 选项 |
|------|------|------|
| 后端类型 | 终端执行环境 | local/docker/modal |
| 默认超时 | 命令执行超时时间 | 秒数 |
| 持久化 Shell | 保持 Shell 会话状态 | 开/关 |

#### 安全设置
| 选项 | 说明 |
|------|------|
| 隐藏敏感信息 | 在日志中自动脱敏 API Key 等 |
| MCP 安全扫描 | 安装插件前检查已知漏洞 |
| Tirith 启用 | 启用策略检查引擎 |

#### 日志与显示
| 字段 | 说明 |
|------|------|
| 日志级别 | DEBUG/INFO/WARNING/ERROR |
| 人格风格 | AI 回复风格（helpful/concise/technical 等） |
| 审批模式 | Manual（手动）/Auto（自动）/Suggest（建议） |

### 操作说明

1. **修改配置**: 直接在表单中编辑需要更改的值
2. **保存配置**: 点击「保存配置」按钮提交更改
3. **重置**: 点击「重置」恢复到当前保存的配置
4. **生效提示**: 部分配置需要重启 Hermes Agent 才能生效

## 3. 对话历史 (History)

**路由:** \`/history\`

对话历史页面提供全量会话的浏览、搜索和管理功能。

### 功能模块

#### 会话列表（左侧面板）
- **FTS5 全文搜索**: 支持中文分词搜索
- **批量选择**: 勾选多个会话进行批量操作
- **批量删除**: 一次性删除多个会话
- 显示会话标题、日期、平台来源

#### 搜索功能
- 输入关键词后自动搜索（300ms 防抖）
- 搜索范围：会话标题、消息内容

#### 会话详情（右侧面板）
- 显示完整的对话消息
- 区分用户消息、Agent 回复、工具调用
- 工具调用显示工具名称和返回内容
- 显示会话 Token 消耗

### 消息类型

| 类型 | 图标 | 说明 |
|------|------|------|
| User | 蓝色用户图标 | 用户发送的消息 |
| Agent | 绿色机器人图标 | Hermes Agent 的回复 |
| Tool | 黄色扳手图标 | 工具调用及返回结果 |

### 操作说明

1. **查看会话**: 点击左侧列表中的会话
2. **搜索历史**: 在搜索框输入关键词
3. **批量删除**: 勾选会话后点击「删除」按钮
4. **导出**: 点击下载按钮导出会话（待实现）

## 4. Skills 管理

**路由:** \`/skills\`

Skills 是 Hermes Agent 的技能库，存储可复用的任务模板和工作流程。

### 功能模块

#### 技能列表（左侧面板）
- **分类过滤**: 按技能类别筛选
- **关键词搜索**: 搜索技能名称、描述、标签
- **文件标识**: 显示技能包含的附加文件类型
  - \`refs\`: 参考文档
  - \`scripts\`: 执行脚本
  - \`templates\`: 配置模板

#### 技能详情（右侧面板）
| 标签页 | 内容 |
|--------|------|
| 说明 | SKILL.md 的完整内容 |
| 参考文档 | 额外的参考资料链接 |
| 脚本 | 预置的执行脚本 |

#### 元信息
- 版本号
- 作者
- 许可证
- 标签列表

### 技能分类

| 类别 | 说明 |
|------|------|
| autonomous-ai-agents | 自主 AI Agent 工具 |
| creative | 创意内容生成 |
| data-science | 数据科学工具 |
| development | 开发辅助工具 |
| devops | 运维自动化 |
| github | GitHub 工作流 |
| mlops | 机器学习运维 |
| productivity | 生产力工具 |

### 操作说明

1. **浏览技能**: 点击列表中的技能查看详情
2. **过滤分类**: 点击顶部分类按钮
3. **搜索技能**: 在搜索框输入关键词

## 5. 定时任务 (Cron)

**路由:** \`/cron\`

定时任务页面用于创建和管理自动化执行的任务。

### 功能模块

#### 任务列表
- 显示所有定时任务
- 任务状态标签：
  - 运行中 (绿色)
  - 已暂停 (黄色)
  - 已完成 (蓝色)

#### 任务卡片信息
| 字段 | 说明 |
|------|------|
| 任务名称 | 自定义任务名 |
| 执行计划 | Cron 表达式或自然语言 |
| 重复进度 | 已执行次数/总次数 |
| 任务提示词 | 任务的具体指令 |
| 绑定技能 | 任务使用的技能列表 |
| 执行脚本 | 预处理脚本 |
| 下次执行 | 计划执行时间 |
| 上次执行 | 最近执行时间 |

### 创建任务

点击「新建任务」打开创建表单：

| 字段 | 必填 | 说明 |
|------|------|------|
| 任务名称 | 否 | 自定义名称，便于识别 |
| 执行计划 | 是 | 支持: \`every 30m\`, \`0 9 * * *\`, ISO 时间戳 |
| 任务提示词 | 是 | 任务的具体指令内容 |
| 重复次数 | 否 | 留空表示无限循环 |
| 交付目标 | 否 | 结果发送位置 (local/telegram/discord) |
| 绑定技能 | 否 | 多个技能用逗号分隔 |

### 调度表达式格式

| 格式 | 示例 | 说明 |
|------|------|------|
| 自然语言 | \`every 30m\` | 每 30 分钟 |
| Cron | \`0 9 * * *\` | 每天 9:00 |
| ISO 时间 | \`2026-01-01T00:00\` | 指定时间一次性执行 |

### 操作说明

| 操作 | 按钮 | 说明 |
|------|------|------|
| 新建 | 「新建任务」 | 打开创建表单 |
| 编辑 | 点击卡片 | 查看详情并编辑 |
| 暂停/恢复 | 播放/暂停图标 | 切换任务状态 |
| 立即执行 | 火箭图标 | 立即触发一次执行 |
| 删除 | 垃圾桶图标 | 删除任务 |

## 6. 系统日志 (Logs)

**路由:** \`/logs\`

系统日志页面提供实时的终端风格日志查看器。

### 功能模块

#### 日志文件选择
- \`agent.log\`: Agent 主程序日志
- \`gateway.log\`: 网关服务日志
- \`cron.log\`: 定时任务日志
- \`mcp.log\`: MCP 插件日志

#### 日志级别过滤

| 级别 | 颜色 | 说明 |
|------|------|------|
| ERROR | 红色 | 错误信息 |
| WARN | 黄色 | 警告信息 |
| INFO | 蓝色 | 一般信息 |
| SUCCESS | 绿色 | 成功信息 |
| DEBUG | 灰色 | 调试信息 |

#### 搜索与控制
- **关键词搜索**: 高亮显示匹配内容
- **行数限制**: 50/100/200/500 行
- **暂停/恢复**: 暂停日志滚动
- **清空**: 清空当前显示的日志

### 日志格式

每条日志包含：
\`\`\`
时间    级别      来源      消息内容
HH:MM:SS [LEVEL] [SOURCE] Message...
\`\`\`

### 操作说明

1. **切换日志文件**: 从下拉菜单选择
2. **过滤级别**: 点击级别按钮开关
3. **搜索日志**: 在搜索框输入关键词
4. **调整行数**: 选择显示的日志行数
5. **暂停滚动**: 点击「暂停」按钮

## 7. 网关状态 (Gateway)

**路由:** \`/gateway\`

网关状态页面监控 Hermes Gateway 的运行状态和平台连接。

### 功能模块

#### 运行状态卡片
| 指标 | 说明 |
|------|------|
| 运行状态 | running/offline/dead |
| 进程 PID | 网关进程 ID |
| 运行时间 | 网关启动以来的时长 |
| 数据大小 | 数据目录占用空间 |

#### 平台连接
显示已配置的消息平台状态：

| 平台 | 说明 |
|------|------|
| 微信 | Weixin/WeChat 网关 |
| Telegram | Telegram Bot |
| Discord | Discord Bot |
| WhatsApp | WhatsApp 网关 |
| Slack | Slack Bot |

连接状态：
- 已连接 (绿色): 正常工作
- 连接中 (黄色): 正在建立连接
- 断开 (红色): 连接失败

#### 已授权用户
- 显示通过配对的用户列表
- 包含用户 ID、平台、授权时间

#### 网关配置
- Gateway 超时设置
- 是否允许所有用户
- 微信消息策略 (open/pairing)

#### 系统状态
- 进程状态（运行中/已停止）
- 端口监听状态
- 日志文件大小

### 操作说明

1. **刷新状态**: 点击「刷新」按钮
2. **查看错误**: 平台卡片会显示错误信息
3. **管理用户**: 查看已授权用户列表

## 故障排除

### Mock 模式
如果页面显示「Mock 模式」，说明：
- \`~/.hermes\` 目录不存在
- Hermes Agent 未正确安装
- 数据显示为模拟数据

### 连接失败
1. 确认 Hermes Agent 正在运行
2. 检查 \`~/.hermes/config.yaml\` 配置
3. 查看系统日志排查错误
