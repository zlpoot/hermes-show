# 功能概览

Hermes Show 是 Hermes Agent 的 Web 可视化管理面板，提供直观的图形界面来管理和监控你的 AI 助手。从系统监控到配置管理，从对话历史到定时任务，这些功能协同工作，让 Hermes 的管理更加高效便捷。

---

## 监控面板

### 仪表盘

仪表盘是系统的核心入口，提供 Hermes Agent 运行状态的全面概览。

**主要功能：**

- **连接状态检测** — 自动检测 `~/.hermes` 目录，显示与本地 Hermes Agent 的连接状态
- **统计卡片** — 实时显示今日 Tokens 消耗、总会话数、今日会话、CPU 负载、活跃 Agent 数量、API 延迟等关键指标
- **Tokens 消耗趋势图** — 近 7 天的 Token 消耗折线图，支持鼠标悬停查看详情
- **活跃任务列表** — 显示当前正在执行的任务，包含任务名称、Agent 类型、平台来源和执行进度
- **最近会话表格** — 快速浏览最近的对话记录，点击可跳转到详情

## 配置管理

### 配置中心

配置中心提供 Hermes Agent 全部设置的图形化管理界面，所有更改保存到 `~/.hermes/config.yaml`。

**模型配置：**

| 字段 | 说明 |
|------|------|
| 当前提供商 | 选择 API 提供商（OpenAI、Anthropic、自定义等） |
| 模型名称 | 指定使用的模型，如 `gpt-4o`、`claude-sonnet-4` |
| API 端点 | 提供商的 API 地址（自定义提供商） |
| 密钥环境变量 | API Key 所在的环境变量名 |

**Agent 设置：**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| Max Tokens | 单次回复最大输出 Token 数 | 8000 |
| Max Turns | 单次任务最大工具调用轮数 | 90 |
| Gateway Timeout | 网关超时时间（秒） | 1800 |
| Reasoning Effort | 推理强度 (low/medium/high) | medium |

**开关选项：**

- **流式输出 (Streaming)** — 启用实时流式响应
- **保存训练轨迹** — 保存对话用于后续训练
- **详细日志 (Verbose)** — 输出更多调试信息

**终端配置：**

| 字段 | 说明 | 选项 |
|------|------|------|
| 后端类型 | 终端执行环境 | local / docker / modal |
| 默认超时 | 命令执行超时时间 | 秒数 |
| 持久化 Shell | 保持 Shell 会话状态 | 开 / 关 |

**安全设置：**

- **隐藏敏感信息** — 在日志中自动脱敏 API Key 等敏感数据
- **MCP 安全扫描** — 安装插件前检查已知漏洞
- **Tirith 启用** — 启用策略检查引擎

**日志与显示：**

| 字段 | 说明 |
|------|------|
| 日志级别 | DEBUG / INFO / WARNING / ERROR |
| 人格风格 | AI 回复风格（helpful / concise / technical / creative 等） |
| 审批模式 | Manual（手动） / Auto（自动） / Suggest（建议） |

## 数据管理

### 对话历史

对话历史页面提供全量会话的浏览、搜索和管理功能。

**主要功能：**

- **FTS5 全文搜索** — 支持中文分词的全文搜索，可搜索会话标题和消息内容
- **批量选择与删除** — 勾选多个会话进行批量删除操作
- **会话详情查看** — 完整显示对话消息，区分用户消息、Agent 回复、工具调用
- **Token 消耗统计** — 每个会话显示消耗的 Token 数量

**消息类型：**

| 类型 | 图标 | 说明 |
|------|------|------|
| User | 蓝色用户图标 | 用户发送的消息 |
| Agent | 绿色机器人图标 | Hermes Agent 的回复 |
| Tool | 黄色扳手图标 | 工具调用及返回结果 |

**使用方式：**

1. 在左侧会话列表中浏览或搜索历史会话
2. 点击会话查看右侧详情面板
3. 使用搜索框输入关键词进行全文搜索
4. 勾选会话后点击「删除」按钮批量删除

### Skills 管理

Skills 是 Hermes Agent 的技能库，存储可复用的任务模板和工作流程。

**主要功能：**

- **分类过滤** — 按技能类别筛选（devops、github、mlops 等）
- **关键词搜索** — 搜索技能名称、描述、标签
- **详情查看** — 查看 SKILL.md 内容、参考文档、执行脚本
- **文件标识** — 显示技能包含的附加文件类型
  - `refs` — 参考文档
  - `scripts` — 执行脚本
  - `templates` — 配置模板

**技能分类：**

| 类别 | 说明 |
|------|------|
| autonomous-ai-agents | 自主 AI Agent 工具 |
| creative | 创意内容生成 |
| data-science | 数据科学工具 |
| development | 开发辅助工具 |
| devops | 运维自动化 |
| github | GitHub 工作流 |
| mlops | 机器学习运维 |
| productivity | 生产力工具 |

## 自动化

### 定时任务

定时任务页面用于创建和管理自动化执行的任务。

**主要功能：**

- **任务创建** — 通过图形界面创建定时任务
- **状态管理** — 暂停、恢复、删除任务
- **立即执行** — 手动触发任务立即运行
- **执行记录** — 查看任务执行历史和输出

**任务配置字段：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 任务名称 | 否 | 自定义名称，便于识别 |
| 执行计划 | 是 | 支持 `every 30m`、Cron 表达式、ISO 时间戳 |
| 任务提示词 | 是 | 任务的具体指令内容 |
| 重复次数 | 否 | 留空表示无限循环 |
| 交付目标 | 否 | local / telegram / discord |
| 绑定技能 | 否 | 多个技能用逗号分隔 |

**调度表达式格式：**

| 格式 | 示例 | 说明 |
|------|------|------|
| 自然语言 | `every 30m` | 每 30 分钟执行 |
| Cron | `0 9 * * *` | 每天 9:00 执行 |
| ISO 时间 | `2026-01-01T00:00` | 指定时间一次性执行 |

**任务状态：**

| 状态 | 颜色 | 说明 |
|------|------|------|
| 运行中 | 绿色 | 任务正在正常执行 |
| 已暂停 | 黄色 | 任务已暂停，不会执行 |
| 已完成 | 蓝色 | 任务已完成所有执行次数 |

## 系统运维

### 系统日志

系统日志页面提供实时的终端风格日志查看器。

**主要功能：**

- **多日志文件切换** — agent.log、gateway.log、cron.log、mcp.log
- **日志级别过滤** — ERROR / WARN / INFO / SUCCESS / DEBUG
- **关键词搜索** — 高亮显示匹配内容
- **实时刷新** — 每 3 秒自动刷新日志
- **暂停/恢复** — 暂停日志滚动以便查看

**日志级别：**

| 级别 | 颜色 | 说明 |
|------|------|------|
| ERROR | 红色 | 错误信息，需要关注 |
| WARN | 黄色 | 警告信息 |
| INFO | 蓝色 | 一般信息 |
| SUCCESS | 绿色 | 成功信息 |
| DEBUG | 灰色 | 调试信息 |

**日志格式：**

```
时间    级别      来源      消息内容
HH:MM:SS [LEVEL] [SOURCE] Message...
```

### 网关状态

网关状态页面监控 Hermes Gateway 的运行状态和平台连接。

**运行状态监控：**

| 指标 | 说明 |
|------|------|
| 运行状态 | running / offline / dead |
| 进程 PID | 网关进程 ID |
| 运行时间 | 网关启动以来的时长 |
| 数据大小 | 数据目录占用空间 |

**平台连接状态：**

| 平台 | 说明 |
|------|------|
| 微信 (Weixin) | Weixin/WeChat 网关 |
| Telegram | Telegram Bot |
| Discord | Discord Bot |
| WhatsApp | WhatsApp 网关 |
| Slack | Slack Bot |

**连接状态：**

- **已连接 (绿色)** — 正常工作
- **连接中 (黄色)** — 正在建立连接
- **断开 (红色)** — 连接失败

**授权用户管理：**

- 显示通过配对的用户列表
- 包含用户 ID、平台来源、授权时间

**网关配置查看：**

- Gateway 超时设置
- 是否允许所有用户
- 微信消息策略 (open / pairing)

## 故障排除

### Mock 模式

如果页面顶部显示「Mock 模式」，说明：

- `~/.hermes` 目录不存在
- Hermes Agent 未正确安装
- 页面显示的数据为模拟数据

**解决方法：**

1. 确保 Hermes Agent 已安装
2. 检查 `~/.hermes` 目录是否存在
3. 运行 `hermes config show` 验证配置

### 连接失败

**排查步骤：**

1. 确认 Hermes Agent 正在运行
2. 检查 `~/.hermes/config.yaml` 配置文件
3. 查看系统日志排查错误信息
4. 确认 API 端点配置正确

### 配置保存失败

**可能原因：**

- 配置文件权限问题
- YAML 格式错误
- 磁盘空间不足

**解决方法：**

```bash
# 检查文件权限
ls -la ~/.hermes/config.yaml

# 手动验证配置
hermes config show
```
